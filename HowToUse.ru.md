# Wirus Mods Manager (WMM) – Руководство пользователя

## 1. Подготовка к работе

1. **Выбор игры и её каталога**  
   - Перейдите в **Настройки → Основное**.  
   - Нажмите **Сменить игру** → выберите нужную игру.  
   - Нажмите **Сменить директорию** → укажите путь к исполняемому файлу игры.  
   - Нажмите **Применить**.  

2. **Создание точки восстановления**  
   - В том же окне нажмите **Создать точку восстановления**.  
   - Это создаст резервную копию данных, которую можно будет восстановить позже.  

## 2. Смена языка интерфейса

1. Перейдите в **Настройки → Язык → Выбрать язык**.  
2. Выберите нужный язык и нажмите **Применить**.  
3. Перезапустите программу, чтобы изменения вступили в силу.  

## 3. Основные действия

| Шаг | Действие               | Что происходит                                                                                                                |
| --- | ---------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| 1   | **Новый объект**       | Открывается диалог создания нового объекта. Введите название и нажмите **Применить**.                                         |
| 2   | **Drag‑and‑drop**      | В нижней части окна есть область «Перетащите файл сюда». Перетащите архив с модом – он будет загружен в базу.                 |
| 3   | **Добавить компонент** | Переключитесь на нужную сборку/пресет, нажмите кнопку, выберите компоненты и подтвердите **Применить**.                       |
| 4   | **Собрать**            | Выберите сборку и нажмите «Собрать». Процесс компиляции запустится. Окно ожидания исчезнет по завершении процесса.            |
| 5   | **Линковка**           | После сборки нажмите «Линковка» – будет начат процесс установки сборки в игру. Окно ожидания исчезнет по завершении процесса. |

## 4. Примечания

- **Пресеты** – объекты, хранящие моду и кортежи. Не могут быть скомпилированы. Созданы для упорядочивания модов в пакетах.  
- **Сборка** – объект, представляющий набор модов. Может содержать пресеты, моду и кортежи. Сборка может быть скомпилирована и установлена в игру. Между переустановками сохраняется кэш сборки.  
- **Кортеж** – объект, состоящий из нескольких версий одного мода. В интерфейсе отображается как версия мода.  
- Чтобы сменить версию мода, нажмите ПКМ по отображаемой версии в сборке.  
- Переключив тумблер «Состояние» объекта в положение **Off**, вы исключаете объект из следующей установки.
